## 核心功能

- **AI 法官「链上包公」** — 综合 DexScreener + Etherscan 真实链上数据，输出 BULLISH/BEARISH 裁定
- **双轨押注** — 用 BNB 或 $GAVEL 押注看涨/看跌，胜方瓜分奖池
- **开庭广场** — 项目方支付 BNB 开庭费，AI 自动分析，社区参与裁决
- **$GAVEL 代币** — BEP-20，持有量决定陪审员等级和权益
- **Meme 雷达** — BSC 实时行情，押注战况可视化
