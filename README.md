## 核心功能

- **AI 法官「链上包公」** — 综合 four.meme 官方 API + DexScreener 真实链上数据，用通俗易懂的语言输出 BULLISH / BEARISH 裁定，不是 chatbot 装饰，是影响押注赔率的经济机制
- **博彩赔率机制** — AI 裁定方向影响赔率权重：AI 看涨时看涨赔率低（最低 1.1x），逆势押跌反而是冷门赔率高（最高 5x）；置信度越高偏移越大
- **多元化盈利渠道** — 不需要持有任何 Meme 币，只需判断对方向就能赚 $GAVEL + BNB；第一次让「判断力」本身成为可变现资产
- **资产对冲机制（行业首创）** — 持有某 Meme 币的同时押注该币「看跌」：币涨了持仓盈利，押注只损失少量 $GAVEL；币跌了押注赢回来部分对冲损失
- **four.meme 自动候审** — 六路并发拉取热度榜 / 涨幅榜 / 新币榜，每 2 分钟刷新，100+ 真实候审案件实时更新
- **三档自动分级** — 系统按市值、持仓数量、涨幅自动分配快审 / 标准 / 深审档位，达标自动升档进入进行中
- **💉 BNB 注入飞轮** — 任何人可为进行中的案件注入 BNB（0.01 BNB = 延长 1 小时），注入全额进奖池，平台不抽佣；奖池越大 → 参与越多 → 注入动机越强 → 社区自发飞轮
- **真实价格结算闭环** — 开庭时记录 openPrice（four.meme 官方价格），倒计时归零时拉取 closePrice，链上对比判定胜负
- **开庭广场** — 全部 / 进行中 / 候审三档 Tab + 快审 / 标准 / 深审筛选 Pills，真实数据实时渲染
- **$GAVEL 代币** — BEP-20，持有量决定陪审员等级和权益
- **Meme 雷达** — DexScreener 真实 K 线图（1秒 / 3分 / 1时三档周期切换），AI 裁定标注直接打在 K 线对应位置
- **「我的」页面** — 收益流水 / 押注 / 奖励 Tab 筛选，胜率战绩卡可视化
- **🥊 打脸记录** — AI 法官历史裁错案例（LUNA 归零、SQUID 跑路、PEPE 涨 7000%），一键截图生成可分享打脸卡片
- **📱 移动端适配** — 独立移动端 UI（≤768px 自动切换）

## 完整运行流程

```
1. four.meme 新币自动进入候审队列
   └── 六路并发 API 每 2 分钟拉取（热度/涨幅/新币三榜各两页）
   └── 自动去重，100+ 候审案件实时更新
   └── 按市值/持仓/涨幅自动分配快审/标准/深审档位

2. 达标自动开庭（或项目方申请开庭）
   └── 记录 openPrice（four.meme 官方实时价格）
   └── 设置倒计时（快审 3h / 标准 24h / 深审 7天）
   └── createCase() 上链

3. 开庭期间
   └── AI 法官分析链上数据，生成通俗语言裁定报告
   └── 裁定方向影响赔率（最高 5x），引导押注策略
   └── 社区用户押注：approve GAVEL → betWithGavel() 两步链上调用
   └── 社区可注入 BNB 延长开庭 + 扩大奖池（飞轮机制）

4. 庭审结束
   └── 拉取 closePrice，与 openPrice 对比
   └── closePrice > openPrice → 看涨方胜；反之看跌方胜
   └── settle() 上链，胜方调用 claimGavel() 领取奖励

5. 新用户激励
   └── 连接钱包后调用 claimNewbie()
   └── 合约自动转账 500 $GAVEL，每地址限一次
```

### V1 结算透明度说明

当前版本（V1）由平台在庭审结束后调用 `settle()`，以 four.meme 官方 API 实时价格为基准。每次结算均公开上链，全程可在 BSCScan 验证。V2 将接入 Chainlink Price Feed，由合约自动读取价格完成结算，彻底去中心化。

## 生态闭环

```
four.meme 新币 ──→ 自动入庭 ──→ AI 裁定开庭
       ↑                                ↓
  项目方付开庭费                   用户押注 $GAVEL + BNB
  获得 AI 背书 + 曝光                    ↓
       ↑                           奖池增大
  社区注入 BNB ←──────────── 吸引更多参与者
  扩大奖池（飞轮）                        ↓
                               胜方赢得奖励 → 再押注
                                           ↓
                                   等级提升 → 更低手续费
```

每一个角色的行为都在强化其他角色的参与动机——真正的生态闭环，而不是单向流量消耗。
